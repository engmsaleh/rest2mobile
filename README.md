R2M Installation
================

Version
-------
 - Stable: 2.3.1

Requirements
------------
  - Java 6 or above

MacOS
-----
Run:
```
brew install https://raw.githubusercontent.com/magnetsystems/r2m/master/r2m.rb
```

If you don't have _brew_, go to: http://brew.sh/


To uninstall r2m:
```
brew remove r2m
```

Other platforms
---------------

Use r2m tar.gz package
